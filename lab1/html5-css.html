HTML5 e CSS

HTML5

HTML5 é uma linguagem de marcação que possui inúmeros elementos semânticos

Elemento semântico Já usei? Descrição
header ✔ Cabeçalhos
nav ✔ Menu de navegação
main ✔ Bloco onde se encontra o conteúdo principal
article ❌ Porpoção do artigo que pode ser visualizada
section ✔ Divisórias de partes do site
aside ❌ Indica anotações, publicidade ou outros assuntos
footer ✔ Rotapés
details ❌ Detalhes, visíveis ou não
meter ❌ Define uma medida escalar estática dentro de um intervalo
progress ✔ Desenvolvimento de barras de progressão
abbr ❌ Significado de uma abreviatura
address ❌ Informação sobre o contacto do autor

CSS

CSS é um mecanismo para adicionar estilos a uma página web

Tipos de seletores Já usei? Descrição
Universal ✔ Aplica o estilo a todos os elementos, utilizado para mudar fontes de texto
Tipo ✔ Aplica o estilo aos elementos de um determinado tipo, utilizado para mudar tamanhos de titulos (h1)
Classe ✔ Aplica o estilo a todos os elementos de uma determinada classe, utilizado para mudar cores de palavras no meio de um texto
Id ❌ Aplica o estilo a um elemento em específico
Atributo ❌ Aplica o estilo a atributos de elementos
Pseudo-classe ✔ Aplica estilos a um estado de um elemento ou conjunto deles, utilizado em botões
Pseudo-elemento ❌ Aplica estilos a elementos em determinado estado
Combinadores de seletores Já usei? Descrição
Seletores descentes (espaço) ✔ Seleciona todos os elementos de um tipo dentro de um div, utilizado para mudar o estilo do parágrafo mas não do titulo
Seletores filhos (>) ❌ Seleciona todos os elementos dentro de um div, mas não dentro de divs dentro desse div
Seletores adjacentes (~) ❌ Seleciona todos os elementos depois de um div
Seletor imediatamente adjacente (+) ❌ Seleciona todos os elementos colocados imediatamente depois de um div
Agrupar (,) ❌ Seleciona todos os elementos dos vários seletores
